# Final project Data Analysis

## Overview
This project analyses [AI-Powered Smart Waste Management System for Urban Cities].

## Research Questions
- Can we predict waste generation based on location features?
- Can clustering help identify zones for optimised waste collection?
- How can AI improve efficiency in urban waste management systems?
- What patterns exist in waste generation across different time periods or locations?
- Can we predict waste generation using historical data?
- How can optimised waste collection reduce operational costs and environmental impact?
- What insights can help cities move toward smarter and more sustainable waste systems?

## Results Summary

This project demonstrates how AI and data analysis can improve urban waste management systems by identifying patterns and enabling smarter decision-making.

### Key Findings:

- Waste generation shows identifiable patterns that can be used for prediction and planning.
- Certain time periods or regions produce significantly higher waste, indicating the need for dynamic collection scheduling.
- Data-driven optimisation can reduce unnecessary waste collection trips, saving fuel and operational costs.
- Predictive insights can help cities allocate resources more efficiently.
- AI-based approaches can contribute to cleaner, more sustainable urban environments.

### Conclusion:
The integration of AI into waste management systems has strong potential to improve efficiency, reduce costs, and support environmental sustainability in modern cities.
