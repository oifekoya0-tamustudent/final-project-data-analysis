# final-project-data-analysis

## Overview
This project analyses [AI-Powered Smart Waste Management System for Urban Cities].

## Research Questions
- Can we predict waste generation based on location features?
- Can clustering help identify zones for optimized waste collection?

## Data
Dataset from: [/content/NYC Waste Management.xlsx]
Preprocessing done in notebooks.

## How to Run
1. Open in Google Colab
2. Install dependencies from requirements.txt
3. Run main_notebook.ipynb

## Dependencies
- Python 3.x
- pandas
- numpy
- scikit-learn
